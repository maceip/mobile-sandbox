>_ Full Linux Terminal on Android

Cory ships a real PTY shell with Python, Node.js, Git, ripgrep, and 60+ Unix tools — bundled into a single APK. No root required.

Download APK View Source

What It Does

🐍

Python 3.14

Full CPython with stdlib, pip, and ensurepip. Run scripts, install packages, build projects.

⬢

Node.js 24

V8-powered JavaScript runtime with npm and npx. Install packages from the registry and run servers.

🦀

Rust FFI

Native Rust staticlib linked via JNI. Extend the runtime with safe, fast native code.

📦

Git

Built from libgit2 with worktree support. Clone, commit, branch, and push from your phone.

🔍

ripgrep

Blazing-fast recursive search. Grep through codebases at native speed on ARM64.

📝

vi / nano

BusyBox vi plus a nano shim. Edit files in the terminal with the extra-keys bar for ESC, CTRL, TAB, and arrows.

🛠️

60+ Unix Tools

BusyBox applets: ls, cat, grep, find, awk, sed, tar, wget, top, and more — all symlinked at boot.

⌨️

IME Extra Keys

Convenience bar above the soft keyboard with ESC, CTRL, TAB, arrows, and PASTE. Inspired by v9's D-pad.
